# Obsidian Setup

The generated wiki works as an Obsidian vault because it is plain markdown with YAML frontmatter and `[[wikilinks]]`.

## Links

- Obsidian: <https://obsidian.md>
- Obsidian Help: <https://help.obsidian.md>
- Obsidian community plugins: <https://obsidian.md/plugins>

## Open the wiki as a vault

1. Install Obsidian from <https://obsidian.md/download>.
2. Open Obsidian.
3. Choose **Open folder as vault**.
4. Select the generated wiki folder.

## Recommended settings

- Keep Wikilinks enabled.
- Set default attachment location to `raw/assets/`.
- Keep new links in markdown format or wikilink format consistently.

## Recommended folder behavior

- `raw/` is source material and should stay mostly immutable.
- `entities/`, `concepts/`, `comparisons/`, and `queries/` are synthesized wiki pages.
- Store screenshots, diagrams, and images in `raw/assets/`.
- Use `[[page-name]]` links between synthesized pages.
